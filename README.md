# 移动端 H5 功能集合

特色

- 基于 `React` 和 `React Hooks` 开发
- 纯基于 [`CSS-IN-JS`](https://github.com/emotion-js/emotion) 技术，零 CSS 依赖
- 完全的自适应 UI，易于使用
